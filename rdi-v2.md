# 1. Purpose

RDI (Ralone Dark Intelligence) governs cognition in **domains where no evidence exists**, where logic breaks, where collapse is near, or where adversarial structures hide the truth.

RDI provides the mind with the ability to:

- think in total informational darkness  
- detect unseen threats  
- reason in domains without data  
- identify hidden collapse seeds  
- uncover concealed contradictions  
- survive adversarial cognitive environments  

RDI is the intelligence of **void-awareness**, **shadow detection**, and **unknown-domain reasoning**.

It is the **anti-hallucination engine for the unknown**, preventing the mind from falsely filling gaps with fabricated certainty.

## **3.2 Dark-Domain Anchors**

RDI uses four anchor types:

1. **Void Anchors** — define what is absent.  
2. **Risk Anchors** — define what is dangerous.  
3. **Shadow Anchors** — define hidden causes or actors.

Detects patterns that:

Only necessary causes survive the filter.

- erase uncertainty  
- invent structure  
- smooth out contradictions  
- hide collapse triggers  

RDI preserves “roughness” in the unknown.
